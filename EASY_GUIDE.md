## 1단계: 데이터 저장소 만들기 (Firebase)

> **쉽게 말하면**: 신도 정보를 저장할 "온라인 금고"를 만드는 거예요

### 🎬 따라하기

**1. Firebase 사이트 접속**
```
https://console.firebase.google.com
```
- 구글 계정으로 로그인하세요

**2. 새 프로젝트 만들기**
```
1. "프로젝트 추가" 버튼 클릭
2. 프로젝트 이름 입력: "해운사관리" (원하는 이름 아무거나)
3. Google 애널리틱스: 끄기 (필요 없음)
4. "프로젝트 만들기" 버튼 클릭
5. 잠시 기다리면 완료! (약 30초)
```

**3. 데이터 저장소 만들기**
```
1. 왼쪽 메뉴에서 "Realtime Database" 클릭
   (못 찾겠으면 "빌드" 메뉴 아래에 있어요)
   
2. "데이터베이스 만들기" 버튼 클릭

3. 위치 선택: "asia-southeast1 (싱가포르)" 선택
   (한국과 가까워서 빨라요)
   
4. 보안 규칙: "테스트 모드에서 시작" 선택
   (나중에 바꿀 수 있어요)
   
5. "사용 설정" 버튼 클릭
```

**4. 중요한 설정 코드 복사하기**
```
1. 왼쪽 상단 톱니바퀴(⚙️) 아이콘 클릭
2. "프로젝트 설정" 클릭
3. 아래로 스크롤해서 "내 앱" 섹션 찾기
4. 웹 아이콘(</>) 클릭
5. 앱 닉네임: "해운사웹" 입력
6. "앱 등록" 버튼 클릭
7. 화면에 나타나는 코드 전체 복사 (메모장에 저장!)
```

**복사할 코드 예시:**
```javascript
const firebaseConfig = {
  apiKey: "AIzaSyABC123...",
  authDomain: "haeunsa-xxxxx.firebaseapp.com",
  databaseURL: "https://haeunsa-xxxxx.firebaseio.com",
  projectId: "haeunsa-xxxxx",
  storageBucket: "haeunsa-xxxxx.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123..."
};
```

⚠️ **이 코드를 메모장에 저장해두세요! 2단계에서 사용합니다!**

## 2단계: 코드 수정하기

> **쉽게 말하면**: 프로그램에 "이 금고를 사용하겠다"고 알려주는 거예요

### 🎬 따라하기

**1. 다운로드한 ZIP 파일 압축 풀기**
```
temple-management-app.zip 파일을 다운로드 폴더에서 찾아서
우클릭 → "압축 풀기" 또는 "여기에 풀기"
```

**2. 메모장으로 파일 열기**
```
1. 압축 푼 폴더 열기
2. "components" 폴더 열기
3. "TempleManagementSystem.jsx" 파일 찾기
4. 우클릭 → "연결 프로그램" → "메모장" 선택
```

**3. Firebase 설정 찾아서 바꾸기**
```
1. Ctrl+F 눌러서 "YOUR_API_KEY" 검색
2. 다음과 같은 부분이 나옵니다:

const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
  ...
};

3. 이 전체 부분을 1단계에서 복사한 코드로 교체!
4. Ctrl+S 눌러서 저장
5. 메모장 닫기
```

**✅ 완료! 이제 2단계 끝입니다!**

## 3단계: 인터넷에 올리기

> **쉽게 말하면**: 만든 프로그램을 인터넷에 올려서 모두가 접속할 수 있게 하는 거예요

### Part A: GitHub (코드 보관함)

**1. GitHub 가입하기**
```
https://github.com 접속
"Sign up" 버튼 클릭하고 가입
(이메일, 비밀번호, 사용자 이름 입력)
```

**2. 새 보관함 만들기**
```
1. 로그인 후 오른쪽 위 "+" 버튼 클릭
2. "New repository" 클릭
3. Repository name: "temple-management" 입력
4. Public 선택 (무료)
5. "Create repository" 버튼 클릭
```

**3. 코드 업로드하기**

**방법 1 - 쉬운 방법 (웹 브라우저 사용)**
```
1. 방금 만든 저장소 페이지에서
2. "uploading an existing file" 링크 클릭
3. 압축 푼 폴더의 모든 파일을 드래그해서 올리기
4. 아래 "Commit changes" 버튼 클릭
```

**방법 2 - 명령어 사용 (조금 복잡)**
```
# Windows: Git Bash 열기 (Git 먼저 설치 필요)
# Mac: 터미널 열기

cd Desktop/temple-management-app
git init
git add .
git commit -m "첫 업로드"
git remote add origin https://github.com/당신의아이디/temple-management.git
git branch -M main
git push -u origin main
```

### Part B: Vercel (인터넷 주소 만들기)

**1. Vercel 가입하기**
```
https://vercel.com 접속
"Sign Up" 버튼 클릭
"Continue with GitHub" 선택 (GitHub 계정으로 로그인)
```

**2. 프로젝트 연결하기**
```
1. "Add New..." 버튼 클릭
2. "Project" 선택
3. GitHub 저장소 목록에서 "temple-management" 찾기
4. "Import" 버튼 클릭
```

**3. 배포하기**
```
1. 설정은 그대로 두고
2. "Deploy" 버튼 클릭
3. 기다리기 (약 2-3분)
4. 완료되면 "Congratulations" 화면이 나옵니다!
```

**4. 주소 확인하기**
```
화면에 다음과 같은 주소가 나타납니다:
https://temple-management-abc123.vercel.app

이 주소를 복사해서 저장하세요!
이것이 바로 여러분의 앱 주소입니다! 🎉
```

## 🆘 문제가 생겼을 때

### "Firebase 연결 안 됨" 오류
```
→ Firebase 설정 코드를 제대로 복사했는지 확인
→ 중괄호 { } 하나라도 빠지면 안 됩니다!
```

### "페이지가 안 열림" 오류
```
→ 2-3분 기다렸다가 다시 시도
→ Vercel 배포가 완료되었는지 확인
```

### "데이터가 저장 안 됨" 오류
```
→ Firebase Console에서 Database가 활성화되었는지 확인
→ Realtime Database 메뉴에 데이터가 보이는지 확인
```
