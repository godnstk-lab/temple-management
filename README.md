# 🙏 해운사 신도관리 시스템

불교 사찰의 신도 정보와 불사 내역을 효율적으로 관리하는 웹 애플리케이션입니다.

## 주요 기능

- 👥 **신도 정보 관리**: 이름, 전화번호, 주소 등 기본 정보 관리
- 📿 **불사 내역 관리**: 불사 내용, 금액, 봉안자 정보 등 상세 기록
- 💰 **입금 내역 추적**: 날짜별 입금 내역 및 미수금 자동 계산
- 🔍 **통합 검색**: 이름, 전화번호, 불사내용으로 빠른 검색
- 🔐 **권한 관리**: 관리자/일반 사용자 구분
- 📱 **반응형 디자인**: PC, 태블릿, 모바일 모두 지원
- 🔄 **실시간 동기화**: Firebase를 통한 다중 사용자 실시간 데이터 공유

## 기술 스택

- **Frontend**: React, Next.js
- **Styling**: Tailwind CSS
- **Database**: Firebase Realtime Database
- **Deployment**: Vercel
- **Icons**: Lucide React

## 시작하기

### 1. 요구사항

- Node.js 18.x 이상
- npm 또는 yarn

### 2. 설치

\`\`\bash
npm install
\`\`\`

### 3. Firebase 설정

1. Firebase 프로젝트 생성
2. Realtime Database 활성화
3. \`components/TempleManagementSystem.jsx\`에서 Firebase 설정 정보 업데이트

### 4. 로컬 실행

\`\`\`bash
npm run dev
\`\`\`

http://localhost:3000 접속

## 사용 방법

### 로그인

- **관리자**: 비밀번호 \`0804\`
  - 신도 추가/수정/삭제 가능
  - 불사 내역 및 입금 내역 관리
  
- **일반 사용자**: 비밀번호 \`1211\`
  - 신도 정보 조회
  - 불사 내역 및 입금 내역 확인

### 주요 작업

1. **신도 추가**
   - 관리자로 로그인
   - "신도 추가" 버튼 클릭
   - 기본 정보 및 불사 정보 입력

2. **불사 내역 관리**
   - 신도 목록에서 "불사내용" 클릭
   - 새 불사 내용 추가 또는 기존 내역 확인

3. **입금 내역 관리**
   - 신도 목록에서 "입금액" 클릭
   - 입금 날짜 및 금액 입력

4. **검색**
   - 상단 검색창에서 이름, 전화번호, 불사내용으로 검색

## 보안

- 비밀번호는 코드에서 관리 (향후 Firebase Authentication으로 업그레이드 권장)
- Firebase Database 규칙으로 읽기/쓰기 권한 제어
- HTTPS를 통한 안전한 데이터 전송
